# CrarK
Is a command line tool designed to brute force encrypted rar files
Programed in C.

## Usage
```
usage: ./CrarK <file> <max_pass_len> <threads>
```
## Installation
```
git clone https://github.com/Ragarr/CrarK
cd CrarK
sudo apt-get install libarchive-dev
make
```
## Example
```
./crack test.zip 4 4
```
